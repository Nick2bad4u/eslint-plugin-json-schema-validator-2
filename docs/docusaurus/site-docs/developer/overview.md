# Developer

Use this section when you need the package's public TypeScript surface rather
than end-user setup guidance.

- Generated API docs describe the plugin entrypoint exports.
- Config docs explain every public Flat Config preset and compatibility alias.
- Rule docs remain the source of truth for user-facing rule options.

The generated API pages are rebuilt from `src/plugin.ts` during the docs build.
